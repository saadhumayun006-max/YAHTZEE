# YAHTZEE

## Description
This is a card game built in Java. The objective is simple: be the first player to get rid of all your cards.

## How to Play

### 1. Setup
* When the application starts, a screen will pop up prompting you to enter the number of rounds you wish to play.

### 2. Gameplay Rules
* Each player is dealt a hand of cards.
* A starting card is placed in the center.
* On your turn, you must place a card from your hand that matches the Rank or Suit of the active card.
* If you cannot play a card, you must draw from the deck.

### 3. Winning
* The game continues until a player has no cards left in their hand.
* The first player to empty their hand wins the round.

## Features
* **Move Tracking:** The game actively tracks and displays the number of moves the player has made during the session.
* **Retry System:** Users have access to a retry button to restart the game immediately or can choose to exit the application.
